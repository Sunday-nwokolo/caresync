# CareSync — Wales Care Management

## Quick Deploy to Vercel

1. Upload this folder to GitHub
2. Connect to Vercel (vercel.com)
3. Deploy — live in 60 seconds

## Database Setup

Run `schema.sql` in your Supabase SQL Editor before first use.

## First Login

After deploying, create your first admin user:
1. Go to Supabase → Authentication → Users → Add User
2. Enter email and password
3. Go to Supabase → Table Editor → staff → Insert row
4. Set role to 'admin', enter same email
